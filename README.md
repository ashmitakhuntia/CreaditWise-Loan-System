# Loan Approval Prediction Using Machine Learning

## Project Overview
This project builds an end-to-end supervised Machine Learning pipeline to predict loan approval status using Binary Classification techniques.
The objective is to analyze applicant data and determine whether a loan should be approved or not.
Most of the cases we see good customers get rejected and suspicios customers get approved to avoid this type of circumstances this project is helpful.
